別端末でのセットアップ手順

Obsidian + Cursor を別端末でも使えるようにする

⏱️ 所要時間：約10分

📌 前提

  • メイン端末で MyVault（Obsidian + Cursor + Git）が構築済み
  • 別端末にも MyVault フォルダは既にある
  • Git の設定（.git）も既にある → トークンの再設定は不要

🔄 同期の仕組み

Obsidian Git プラグインにより、5分ごとに自動で GitHub に同期されます。手動操作は不要です。

🏠
メイン端末
Obsidian
⟷ 自動 ⟷
🐙
GitHub
⟷ 自動 ⟷
💻
別端末
Obsidian
✅ Obsidian を開いているだけで自動同期
・起動時：自動で git pull（最新を取得）
・5分ごと：変更があれば自動で git commit + push
・ターミナル操作は一切不要

📊 何が同期されるか

項目同期備考
ノート・フォルダ✅ 自動00_Inbox、20_Context、30_Projects など
Obsidian設定・プラグイン✅ 自動.obsidian/ も含まれる
Cursorルール✅ 自動.cursorrules
Cursorエディタ設定❌フォント・テーマ等は端末ごと
Cursorアカウント❌端末ごとにログインが必要
STEP 1

🛠️ ツールのインストール

1
Git git-scm.com を開く →「Download for Windows」（Macなら「Download for macOS」）をクリック → ダウンロードしたファイルを開いてインストール。
設定はすべて「Next」→「Install」でOK。変更不要。
2
Obsidian obsidian.md を開く →「Get Obsidian for free」→ ダウンロード・インストール。
3
Cursor cursor.com を開く →「Download」→ ダウンロード・インストール。
STEP 2

📂 Obsidian で MyVault を開く

1
Obsidian を起動する
初回起動すると「Vault を開くか作るか」の画面が出ます。
Open folder as vault（既存のフォルダを保管庫として開く）をクリック。

※ 日本語の場合は「フォルダーをボルトとして開く」と表示される場合もあります
2
MyVault フォルダを探して選択する
フォルダ選択ダイアログが開きます。以下の場所にあります：

🪟 Windows の場合：
左のナビゲーションで ドキュメント をクリック → Documents フォルダを開く → MyVault を選択 → フォルダーの選択

パスで言うと：
C:\Users\（あなたの名前）\OneDrive\ドキュメント\Documents\MyVault

🍎 Mac の場合：
Finder で OneDrive → ドキュメント → Documents → MyVault を選択

⚠️ MyVault フォルダ自体を選択してください（中のファイルではなく）
3
プラグインを有効化する
初めて開くと「このVaultのプラグインを信頼しますか？」と聞かれます。
Trust author and enable plugins をクリック。

※ これにより Obsidian Git（自動同期）プラグインが有効になります
4
正しく開けたか確認
左サイドバーに以下のフォルダが表示されていればOK：
📁 00_Inbox 📁 10_Goals 📁 20_Context 📁 30_Projects ...

画面右下のステータスバーに ✓ アイコンが表示されていれば、Git同期も正常に動作しています。
STEP 3

🤖 Cursor で MyVault を開く

1
Cursor を起動してログインする
初回起動時にログイン画面が表示されます。
Sign in → Continue with Google をクリック。
メイン端末と同じアカウントでログインすると、Proプランが共有されます。
2
MyVault フォルダを開く
画面上部メニューの File → Open Folder をクリック。
STEP 2 と同じ MyVault フォルダを選択して「開く」。
3
AI が正しく動作するか確認する
チャットパネルを開きます：
Windows: Ctrl + L　/　Mac: Cmd + L

以下をチャットに入力して送信：
テスト：.cursorrules が読み込めているか確認して
→ AI がVault構造やルールについて回答すれば セットアップ完了 です 🎉
⚠️ 重要：ファイルではなくフォルダを開く
.cursorrules は MyVault フォルダを「Open Folder」で開いたときだけ読み込まれます。
ファイルを1つずつ開いても AI ルールは適用されません。

✅ セットアップ完了チェック

  • Git・Obsidian・Cursor がインストールされている
  • Obsidian で MyVault を開いてフォルダ構造が見える
  • Obsidian のプラグインを有効化した
  • Obsidian 右下に Git 同期のアイコンが表示されている
  • Cursor で MyVault フォルダを開いている
  • Cursor チャットで AI が .cursorrules を認識している

🔧 トラブルシューティング

Obsidian でフォルダ構造が表示されない
MyVault フォルダではなく、別のフォルダを開いている可能性。
Obsidian 左下の 🏠 アイコン →「別のVaultを開く」→ 正しい MyVault フォルダを選択。
Obsidian 右下に Git アイコンが表示されない
① プラグインの有効化を確認：設定（⚙️）→「コミュニティプラグイン」→「制限モード」がオフ → Obsidian Git がオンか確認
② Git がインストールされているか確認（STEP 1）
③ Obsidian を再起動
「認証エラー」が出る（Git同期失敗）
PAT（Personal Access Token）の期限切れの可能性。
GitHub → Settings → Tokens で確認。
再発行が必要な場合は Cursor のターミナル（Ctrl + `）で：
git remote set-url origin https://ユーザー名:新PAT@github.com/ユーザー名/MyVault.git
Cursor で AI ルールが効かない
MyVault フォルダ自体を「Open Folder」で開いているか確認。
ファイル単体を開いてもルールは適用されません。
同期が競合した（conflictと表示される）
2つの端末で同時にObsidianを開いていた場合に発生することがあります。
→ 片方のObsidianを閉じる → もう片方で Obsidian Git の「Pull」ボタンを押す → 競合ファイルを確認して修正。
→ 基本は 同時に2つの端末で開かない ことで防げます。
